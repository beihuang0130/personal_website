# 图片上传说明

## 📸 请在此文件夹上传以下图片：

### 头像图片
- **文件名**: `avatar.jpg`
- **建议尺寸**: 400x400 像素（正方形）
- **格式**: JPG, PNG 或 WebP
- **用途**: 首页展示的个人头像

### 项目截图
请在 `projects/` 子文件夹中上传项目截图：
- **文件名**: `project1.jpg`, `project2.jpg`, `project3.jpg` 等
- **建议尺寸**: 1200x800 像素（3:2 比例）
- **格式**: JPG, PNG 或 WebP
- **用途**: 项目作品展示

## 💡 提示
- 图片文件名需要与 `js/main.js` 中的配置一致
- 建议压缩图片以提高网页加载速度
- 支持的图片格式：.jpg, .jpeg, .png, .webp, .gif
