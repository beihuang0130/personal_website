### 1. 修改个人信息（5分钟）
📁 打开文件：`js/main.js`

找到第3-50行的 `personalData` 对象，修改：
- [ ] `name`: 你的名字
- [ ] `title`: 你的职业
- [ ] `bio`: 个人简介
- [ ] `email`: 邮箱地址
- [ ] `location`: 所在地
- [ ] `social.github`: GitHub链接
- [ ] `social.linkedin`: LinkedIn链接

### 2. 上传头像（2分钟）
- [ ] 准备一张头像图片（400x400像素）
- [ ] 重命名为 `avatar.jpg`
- [ ] 放入 `images/` 文件夹

### 3. 添加技能（3分钟）
📁 继续在 `js/main.js` 中

修改 `skills` 数组（第30-37行左右）：
```javascript
skills: [
    { name: "你的技能1", icon: "💻", level: 90 },
    { name: "你的技能2", icon: "🎨", level: 85 },
    // 添加更多...
]
```

常用emoji图标：💻 🎨 📱 ⚙️ 📊 🚀 🎯 ✨ 🔧 📝

### 4. 添加项目（10分钟）
修改 `projects` 数组（第40-60行左右）：

```javascript
projects: [
    {
        name: "项目名称",
        description: "项目描述",
        image: "images/projects/project1.jpg",
        link: "项目链接",
        tags: ["技术1", "技术2"]
    }
]
```

然后上传项目截图：
- [ ] 准备项目截图（1200x800像素）
- [ ] 重命名为 `project1.jpg`, `project2.jpg` 等
- [ ] 放入 `images/projects/` 文件夹

### 5. 本地预览（1分钟）

- [ ] 检查所有信息是否正确
- [ ] 测试主题切换按钮
- [ ] 测试响应式（按F12，切换设备视图）

## 🎨 可选：自定义主题颜色

📁 打开文件：`css/style.css`

找到第4-15行的 `:root` 部分，修改颜色：

```css
:root {
    --accent-primary: #6366f1;      /* 主色调 */
    --accent-secondary: #8b5cf6;    /* 次色调 */
}
```

**推荐配色**：
- 蓝紫色（默认）：`#6366f1` + `#8b5cf6`
- 粉红色：`#ec4899` + `#f43f5e`
- 绿色：`#10b981` + `#14b8a6`
- 橙色：`#f59e0b` + `#f97316`
